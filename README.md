# Class Productivity Analytics

## Project Description

This project analyzes class productivity data using Python and Jupyter Notebook.

## Repository Structure


Class-Productivity-Analytics/
│
├── data/
│   └── class_productivity_data.csv
│
├── notebooks/

│
├── visuals/
│   ├── chart_1.png
│   ├── chart_2.png
│   ├── chart_3.png
│   └── additional_charts.png
│
├── README.md
└── requirements.txt


## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## How to Run

1. Install the required libraries.
2. Open the notebook.
3. Run all cells.
4. View the generated charts.
